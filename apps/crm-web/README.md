# CRM Web Uygulaması

Bu klasör mevcut PHP CRM uygulamasının monorepo içindeki uygulama alanıdır.

İlk aşamada mevcut CRM davranışı değiştirilmeden buraya taşınacaktır.

## Kapsam

- Dashboard
- Kullanıcılar
- Bayi / Firma
- Takip listesi
- Satış fırsatları
- Raporlar

## Not

Bu klasör şu an iskelet olarak oluşturuldu. Uygulama taşıma adımı ayrı checklist maddesidir.
