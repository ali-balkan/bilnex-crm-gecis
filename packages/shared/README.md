# Shared Paketi

CRM içinde ortak kullanılan sabitler ve yardımcılar burada tutulacaktır.

## Hedef İçerik

- CustomerTypeId eşleştirmeleri
- rol sabitleri
- form validasyon yardımcıları
- tarih ve para formatlama yardımcıları

## CustomerTypeId Kararları

- İş ortağı: `7`
- Hedef bayi: `14`
- Müşteri: `16`
- Hedef müşteri: `17`
